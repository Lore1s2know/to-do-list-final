# To-Do List

A fully functional to-do list built with vanilla JavaScript, Tailwind CSS, and rainbow-style colors. Tasks are saved in the browser with `localStorage`.

## Features

- Add, complete, edit, and delete tasks
- Filter by All, Active, or Completed
- Clear completed tasks
- Persistent storage in the browser
- No build step or Node.js required

## Run locally

Open `index.html` in your browser. No install or server needed.

## Deploy on GitHub Pages

This project is a static site and is ready for GitHub Pages.

### One-time setup

1. Push this repository to GitHub.
2. Open **Settings → Pages** in the repository.
3. Under **Build and deployment**, set **Source** to **GitHub Actions**.
4. Push to `main`, or run the **Deploy to GitHub Pages** workflow manually from the **Actions** tab.

The workflow uploads the repository root as-is. It does not use Node.js or a build step.

### Live URL

After deployment, the site will be available at:

`https://lore1s2know.github.io/to-do-list-final/`

Replace the username or repository name if your GitHub repo differs.
